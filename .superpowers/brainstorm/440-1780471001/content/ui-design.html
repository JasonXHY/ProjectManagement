✅ UI设计要点

• 简约现代：大量留白，简洁线条，类似Notion、Linear风格

• 清新自然：柔和色彩，舒适视觉体验

• 功能导向：清晰层次结构，易于操作

• 响应式设计：适配不同屏幕尺寸

• 一致性：统一的组件样式和交互模式

💡 技术实现

• 组件库：Ant Design X（对话组件） + shadcn/ui（轻量组件）

• 样式方案：CSS Modules + Tailwind CSS

• 图标：Ant Design Icons + 自定义SVG

• 主题：支持亮色/暗色主题切换
