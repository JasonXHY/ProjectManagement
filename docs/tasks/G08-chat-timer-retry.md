# G8 — 对话「已等待 X 秒」+ retry 上限 2

> **状态：✅ 完成（TDD，commit `71df9ee`，分支 feat/v3.1-subcategories-and-fixes）**

> 关联需求：F05.2
> 依赖：无（独立）
> 阻塞：无
> 优先级：P3

## 背景

F05.2 要求：AI 回复过程中显示"已等待 X 秒..."；回复失败时显示"重试"按钮且**最多 2 次重试**；发送按钮在回复中变为红色方形"停止"图标。当前 `ChatWindow.tsx`：只有打字点动画（无秒数计时器）；retry 仅把内容重新填回输入框、无 max-2 计数；stop 是文字"取消"而非红方块图标。

## Scope

### 包含
1. 计时器：AI 请求发出后启动秒级计时，在等待区域显示"已等待 X 秒..."，回复返回或取消时停止/清零。
2. 重试逻辑：失败后展示"重试"，内部维护重试计数，**达到 2 次后禁用重试**并提示。
3. 停止按钮：发送按钮在回复期间切换为红色方形"停止"图标（样式对齐 spec）。

### 不包含
- 流式输出（spec 未要求 token 流）。
- 后端 AI 调用改动。

## 成功标准
- [ ] 等待期间可见"已等待 X 秒..."并随时间递增
- [ ] 失败后可重试，最多 2 次，超限禁用并提示
- [ ] 回复期间发送按钮变为红色方形停止图标，点击可中止
- [ ] 计时器在卸载/会话切换时正确清理（无泄漏，沿用 B6/B9 约定）
- [ ] 组件测试（fake timers）：秒数递增；重试计数达 2 后禁用
- [ ] `npm run typecheck && npm run lint && npm test` 全绿
