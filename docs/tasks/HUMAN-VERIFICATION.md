## 如何启动
```bash
npm install
npm run electron:compile   # 必须先编译主进程
npm run electron:dev       # 启动 Vite + Electron
```
> ⚠️ AI 相关项需先在「设置 → AI模型」配置一个有效厂商 API Key（如智谱 GLM 免费档）。未配置时，项目/文件管理、设置 UI、文件夹结构、排序均可用，仅实时 AI 调用会报错。
