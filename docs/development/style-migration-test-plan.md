# 样式迁移测试方案

> 版本：1.0 | 日期：2026-06-14

---

## 一、迁移目标

将组件内联样式迁移到CSS变量，保持视觉一致性。

### 可迁移的样式
- 颜色（`color`, `background`）
- 圆角（`borderRadius`）
- 阴影（`boxShadow`）
- 间距（`padding`, `margin`, `gap`）
- 字体大小（`fontSize`）
- 边框（`border`）

### 保留为内联的样式
- 动态样式（基于状态/props的颜色切换）
- 动画相关样式
- 布局相关（flex, grid, width, height）
- 响应式样式

## 四、回滚策略

如果迁移后出现问题：
1. 保留Git提交记录
