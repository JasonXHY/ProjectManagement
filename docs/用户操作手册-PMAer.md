# PMAer 用户操作手册

> 版本：v0.2.0 | 更新日期：2026-06-29

---

## 一、简介

PMAer（Project Management Assistant）是一款AI驱动的项目管理桌面应用。核心理念：

- **文件智能分类**：上传项目文件后，AI自动按10个阶段+子分类归档，告别手动整理
- **AI提取关键信息**：自动识别合同号、客户信息、关键问题等核心内容
- **项目绑定对话**：AI对话绑定在项目上，Agent能直接阅读你的项目文件，给出针对性建议

---

## 二、安装与启动

1. 双击 `PMAer-0.2.0-Setup.exe`，按提示选择安装目录
2. 安装完成后桌面生成快捷方式，双击启动
3. 首次启动会弹出引导窗口：
   - **Step 1**：选择项目文件存储位置（默认在安装目录下的 `projects` 文件夹，可自定义）
   - **Step 2**：选择AI配置方式
     - **使用内置Key**：开箱即用，已预置免费AI接口（2026-07-31到期）
     - **使用自己的Key**：填写自己的API Key，支持11家AI厂商

---

## 三、主界面

应用分为4个主要页面：

| 页面 | 说明 |
|------|------|
| 项目列表 | 所有项目的概览，支持创建、搜索、删除 |
| 项目首页 | 文件管理、AI分类、阶段推进、⭐卡片信息面板 |
| AI对话 | 与Agent对话，Agent可读取项目文件 |
| 设置 | AI模型、文件提取、Prompt、角色等配置 |

---

## 四、项目管理

### 4.1 创建项目

1. 在项目列表页点击「新建项目」
2. 输入项目名称
3. 选择分类方式：
   - **按阶段分类**：文件按10个阶段（售前→关闭）自动归档，适合项目全生命周期管理
   - **按内容分类**：文件按内容类型（文档、代码、表格等）自动归类
4. 点击确认，系统自动创建项目文件夹结构

### 4.2 项目阶段

每个项目有3个**项目阶段**（不可自定义）：

```
售前 → 进行中 → 关闭
```

阶段推进方式：
- **手动推进**：在项目首页点击「推进到下一阶段」按钮
- **自动推进**：上传文件时，AI判断文件属于更高阶段会自动弹窗确认推进

## 五、文件管理

### 5.1 上传文件

在项目首页有3种上传方式：

1. **拖拽上传**：将文件直接拖到上传区域
2. **点击上传**：点击上传区域选择文件
3. **侧边栏上传**：点击左侧底部的「上传文件」按钮

支持格式：PDF、Word（.doc/.docx）、Excel、PPT、TXT、Markdown、图片
单文件限制：50MB

### 5.2 AI自动分类

文件上传后，AI自动完成3项判断：

| 判断 | 说明 | 示例 |
|------|------|------|
| 文件分类阶段 | 属于10个阶段中的哪个 | 需求、测试、上线 |
| 子分类 | 属于该阶段下的哪个用途 | 需求文档、会议纪要 |
| 项目阶段 | 属于售前/进行中/关闭 | 进行中 |

分类完成后，文件自动移动到对应文件夹。

### 5.3 手动分类

- **单个文件**：在文件列表中点击「重新分类」按钮
- **批量分类**：勾选多个文件，点击顶部「批量分类」按钮
- **一键分类**：点击「一键分类」按钮，对所有未分类文件批量触发AI分类

### 5.4 10个文件分类阶段

| 阶段 | 子分类 |
|------|--------|
| 售前 | 销售方案、报价单、合同原件、客户沟通、成本评估、POC材料 |
| 启动 | 项目章程、团队组建、启动会议 |
| 需求 | 需求文档、会议纪要、需求变更、客户材料、项目计划 |
| 方案 | 开发规格说明书、蓝图、方案汇报材料、会议纪要 |
| 构建 | 开发文档、接口文档、配置文档 |
| 测试 | 测试用例、测试报告、测试数据、会议纪要 |
| 上线 | 上线切换方案、部署文档、操作手册、签字材料、初始化材料、问题清单、会议纪要 |
| 验收 | 验收材料待签、验收签字件已签、验收报告、项目总结 |
| 转客户成功 | 交接文档、培训资料、FAQ |
| 关闭 | 项目归档、复盘总结 |

### 5.5 其他操作

- **打开文件夹**：点击「打开文件夹」按钮，直接在资源管理器中查看文件
- **删除文件**：在文件列表中点击删除按钮
- **刷新列表**：点击「刷新」按钮重新加载文件

## ⭐ 六、项目卡片信息面板（v0.2.0新增）

项目首页上方显示统计行和特色卡片，帮助你快速掌握项目全貌。

### 6.1 统计行

统计行固定显示4张小卡片：

| 卡片 | 显示内容 | 点击操作 |
|------|---------|---------|
| 文件数量 | 项目文件总数 | 无 |
| 里程碑 | 下一步里程碑名称和日期 | 弹窗查看完整时间轴 |
| 待处理 | 未被AI分析的文件数 | 无 |
| AI摘要 | 查看/生成项目摘要 | 弹窗查看或生成摘要 |

### 6.2 特色卡片

卡片根据项目阶段动态切换：

**售前阶段**显示：
- 📋 项目信息：联系人、合同号、电话等基本信息
- 📄 合同概览：合同总额、已确认收入、合同分项、付款里程碑
- 🖥️ 项目评估：⭐利润测算工具（详见§6.3）

**进行中阶段**显示：
- 📝 需求跟踪：需求列表，支持行内展开查看方案和时间轴
- ⚠️ 关键问题：问题清单，支持勾选标记已解决
- ✏️ 签字追踪：签字文件状态和进度

**关闭阶段**显示：
- 🚀 拓展商机：二期/三期规划和商机
- 📦 交付物清单：交付物状态和版本历史
- 📊 项目总结：项目成果和复盘，支持复制为汇报格式

### ⭐ 6.3 利润测算工具

点击「项目评估」卡片中的「利润测算」链接，打开利润测算弹窗。

**使用步骤**：
1. 输入合同总额（如有合同文件，系统自动填入）
2. 添加团队成员：点击「+ 添加成员」，选择角色和职级，输入人天
3. 填写外包信息：外包人天、外包单价、外部差旅
4. 填写内部差旅
5. 系统实时计算利润率

**查看结果**：
- 成本分配明细：内部/外包人天成本、差旅、总成本
- 利润率三栏：内部/外包/整体利润率，带进度条
- 红线警告：内部利润率<0%或外包利润率<40%时自动提醒

**保存和复制**：
- 点击「保存」按钮，测算结果自动保存到项目
- 点击「复制结果」按钮，一键复制测算文本到剪贴板
- 下次打开弹窗，自动恢复上次的测算数据

## 七、AI对话

### 7.1 进入对话

在项目首页点击右上角的「对话」按钮进入AI对话窗口。

### 7.2 对话特点

- **项目绑定**：对话绑定在当前项目上，AI能读取项目中的所有文件
- **多会话管理**：支持创建多个对话会话，每个会话独立记录
- **文件上下文**：点击右侧面板的📎按钮，选择文件作为对话上下文，AI会基于文件内容回答

### 7.3 对话操作

| 操作 | 说明 |
|------|------|
| 发送消息 | 输入框输入内容，点击发送或按Enter |
| 新建会话 | 点击左侧「+」按钮 |
| 切换会话 | 点击左侧会话列表 |
| 选择文件 | 点击📎按钮展开文件面板，勾选文件作为AI上下文 |
| 重试 | AI回复失败时显示重试按钮 |
| 停止 | AI回复中可点击停止按钮中断 |

### 7.4 AI摘要生成

在项目首页统计行点击「生成/更新」按钮，AI会基于项目所有文件生成结构化摘要，包含：
- 项目概述
- 文件清单
- 当前进展
- 关键问题
- 建议和风险

生成后点击「查看摘要」按钮，以Markdown格式渲染展示。

## ⭐ 八、项目转交（v0.2.0新增）

转交功能用于将项目打包分享给同事，或从同事处导入项目。

### 8.1 导出转交包

1. 在项目首页点击「转交」按钮
2. 选择转交模式：
   - **整体转交**：打包整个项目的所有文件和分析数据
   - **选择性转交**：勾选需要转交的文件，或让AI智能推荐
3. 填写转交说明（可选）
4. 点击「导出」，选择保存位置，生成`.pmaer.zip`文件

### 8.2 导入转交包

1. 在项目列表页点击「导入」按钮
2. 选择`.pmaer.zip`文件
3. 预览项目信息：名称、文件数量、阶段
4. 可修改导入后的项目名称
5. 点击「确认导入」

**导入后保留的数据**：
- ✅ 文件分类和子分类
- ✅ AI分析摘要和关键信息
- ✅ 项目阶段
- ✅ 签字状态
- ✅ 项目元数据（合同金额、客户信息等）

## 十、设置

点击右上角⚙️进入设置页面，包含6个配置项：

### 10.1 AI模型

| 配置项 | 说明 |
|--------|------|
| AI厂商 | 选择AI服务提供商（小米MiMo、智谱、阿里千问、DeepSeek等11家） |
| AI模型 | 选择具体模型 |
| API Key | 填写API密钥。内测版已内置，可点击「使用自己的Key」切换 |
| 计费模式 | 按量计费 / Token Plan / 自定义URL |
| API地址 | 自定义API接口地址（高级） |

### 10.2 文件提取

配置各格式文件的提取方式（本地/云端）：
- 纯文本（.txt）：本地提取
- PDF文本层：本地提取
- PDF扫描件：云端OCR
- Word文档：本地提取（支持.doc/.docx）
- Excel表格：本地提取
- 图片：云端OCR

### 10.3 存储设置

| 配置项 | 说明 |
|--------|------|
| 项目文件存储位置 | 所有项目文件的根目录，支持浏览选择 |

### 10.4 Prompt配置

可自定义AI使用的Prompt模板：
- 文件分类Prompt（按阶段）
- 文件分类Prompt（按内容）
- 项目分析Prompt

点击「恢复默认」可还原为内置Prompt。

### 10.5 用户角色

选择你的角色（如项目经理、开发工程师等），AI会根据角色调整回复风格和关注点。

### 10.6 文件分类管理

自定义10个分类阶段的顺序和子分类：
- 拖拽调整阶段顺序
- 新增/删除子分类
- 每个子分类最多50个字符

## 十二、常见问题

### Q：安装后提示"AI服务未配置"怎么办？

打开设置页 → AI模型，选择厂商并填写API Key。内测版可直接选择「使用内置Key」。

### Q：文件上传后没有自动分类？

检查AI是否已配置。未配置AI时，系统会根据文件名推断分类（准确度较低）。

### Q：如何修改已分类的文件？

在文件列表中找到该文件，点击「重新分类」按钮，AI会重新判断分类。

### Q：项目阶段可以倒退吗？

不支持。阶段只能从售前→进行中→关闭单向推进，防止误操作。

### Q：内置API Key什么时候到期？

2026年7月31日到期。届时请在设置页切换到自己的API Key。

### Q：支持哪些文件格式？

PDF、Word（.doc/.docx）、Excel、PPT、TXT、Markdown、JPG/PNG图片。单文件最大50MB。

### Q：利润测算的数据会丢失吗？

不会。测算结果自动保存到项目中，下次打开弹窗会恢复上次的数据。

### Q：转交导入后需要重新分析文件吗？

不需要。导入会保留原有的文件分类、AI摘要、阶段信息等所有分析数据。
